### Development Setup Process

_Node and NPM or Yarn must be installed_

```bash
Installing Dependencies for both client and server

Client

- npm install

Server

- cd server

- npm install

Running Client

- npm run dev:client

Running Server

- npm run server

Running Together

- npm run start
```
